
# ЧЕК-ЛИСТ функционального тестирования

---

## 1. Юридическая проверка авто по VIN-номеру

- [ ] Поле VIN присутствует на экране
- [ ] Поле принимает ровно 17 символов
- [ ] Поле не принимает менее 17 символов
- [ ] Поле не принимает более 17 символов
- [ ] Поле не принимает буквы I, O, Q (запрещены в VIN)
- [ ] Поле принимает латинские буквы и цифры
- [ ] Поле не принимает спецсимволы
- [ ] При вводе валидного VIN результат проверки отображается корректно
- [ ] При вводе невалидного VIN отображается сообщение об ошибке
- [ ] Результат содержит юридическую информацию об автомобиле
- [ ] Кнопка «Проверить» неактивна при пустом поле
- [ ] Работает при отсутствии интернета (отображается корректная ошибка)

---

## 2. Калькулятор алкоголя

- [ ] Форма калькулятора отображается корректно
- [ ] Присутствуют поля: пол, вес, тип напитка, объём, крепость
- [ ] Поле веса принимает только числовые значения
- [ ] Поле веса не принимает отрицательные значения и ноль
- [ ] Выбор типа напитка работает корректно (выпадающий список / кнопки)
- [ ] Поле объёма принимает только числовые значения
- [ ] Поле крепости принимает только числовые значения
- [ ] Результат расчёта отображается после заполнения всех полей
- [ ] Результат пересчитывается при изменении данных
- [ ] Отображается время выведения алкоголя из организма
- [ ] При незаполненных обязательных полях отображается ошибка
- [ ] Отображается предупреждение о том, что результат приблизительный

---

## 3. Заказ эвакуатора

- [ ] Кнопка «Заказать услугу» присутствует на главном экране
- [ ] Выпадающий список открывается по нажатию
- [ ] Пункт «Вызов эвакуатора» присутствует в списке
- [ ] Форма открывается после выбора услуги
- [ ] Присутствуют поля: Имя, Телефон, Email
- [ ] Поле «Имя» не принимает цифры и спецсимволы
- [ ] Поле «Телефон» принимает корректный формат номера
- [ ] Поле «Телефон» не принимает буквы
- [ ] Поле «Email» валидирует корректный формат (наличие @ и домена)
- [ ] Кнопка «Заказать» неактивна при незаполненных полях
- [ ] После отправки появляется модальное окно с подтверждением
- [ ] Модальное окно закрывается корректно
- [ ] Заявка не отправляется повторно при повторном нажатии кнопки

---

## 4. Генератор документов

- [ ] Раздел «Генератор документов» открывается корректно
- [ ] Присутствует список доступных типов документов
- [ ] Выбор типа документа работает корректно
- [ ] Все обязательные поля формы присутствуют
- [ ] Обязательные поля валидируются перед генерацией
- [ ] При незаполненных полях отображается ошибка
- [ ] Документ генерируется после заполнения всех полей
- [ ] Сгенерированный документ доступен для скачивания / просмотра
- [ ] Формат документа корректный (PDF / DOCX)
- [ ] Данные пользователя корректно подставляются в документ
- [ ] Повторная генерация с теми же данными даёт идентичный результат

---

## 5. Запись на ТО

- [ ] Раздел «Запись на ТО» открывается корректно
- [ ] Присутствует выбор даты и времени
- [ ] Недоступные даты заблокированы в календаре
- [ ] Прошедшие даты недоступны для выбора
- [ ] Присутствует выбор сервисного центра / мастера
- [ ] Присутствуют поля: Имя, Телефон, марка/модель авто
- [ ] Поле телефона валидируется корректно
- [ ] Кнопка «Записаться» неактивна при незаполненных полях
- [ ] После отправки отображается подтверждение записи
- [ ] Подтверждение содержит дату, время и адрес сервиса
- [ ] Запись отображается в истории / личном кабинете пользователя
- [ ] Возможность отменить запись

---

## 6. Построение схемы ПДД

- [ ] Раздел «Схема ПДД» открывается корректно
- [ ] Инструменты для построения схемы присутствуют на экране
- [ ] Добавление элементов дороги (полосы, перекрёстки) работает корректно
- [ ] Добавление знаков ПДД работает корректно
- [ ] Добавление участников движения (авто, пешеходы) работает корректно
- [ ] Элементы можно перемещать по схеме
- [ ] Элементы можно удалять
- [ ] Отмена последнего действия (Undo) работает корректно
- [ ] Схема масштабируется (zoom in / zoom out)
- [ ] Готовую схему можно сохранить
- [ ] Готовую схему можно скачать / экспортировать (PNG, PDF)
- [ ] Сохранённая схема открывается корректно при повторном входе
